# inotify-cho

inotify-cho is a service that changes group ownership of a file or directory moved to a watched directory.

## Getting Started

### Prerequisites

* [inotify-tools](https://github.com/rvoicilas/inotify-tools/wiki)

### Install

```bash
sudo bash installer.sh
```
